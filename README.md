# 📊 CDP 逆市操作指标

基于昨日（最新交易日）最高价、最低价、收盘价计算 CDP 经典日内支撑/阻力位，并提供 A 股实时数据获取、快捷选股存储功能，适用于短线逆市交易参考。

## ✨ 功能特性

- **自动获取最新数据**：输入 A 股代码（如 600519、000858），一键获取最近一个交易日的高、低、收数据。
- **CDP 关键位计算**：实时计算 AH（最高阻力）、NH（近高阻力）、CDP（中枢值）、NL（近低支撑）、AL（最低支撑）。
- **快捷选股区**：最多保存 10 只常用股票，自动获取股票名称，点击即可加载数据，免除重复输入。
- **自动重算**：修改任意价格字段，CDP 点位立即重新计算。
- **本地存储**：快捷股票列表保存在浏览器 localStorage 中，关闭页面不丢失。

## 🧮 CDP 计算公式

- **CDP** = (H + L + 2×C) / 4  
- **AH** (最高阻力) = CDP + (H - L)  
- **NH** (近高阻力) = 2×CDP - L  
- **NL** (近低支撑) = 2×CDP - H  
- **AL** (最低支撑) = CDP - (H - L)

> 其中 H、L、C 分别为前一交易日的最高价、最低价、收盘价。

## 📈 逆市操作参考

| 市场状态 | 价格位置 | 操作建议 |
|---------|---------|----------|
| 箱体震荡 | 价格在 NL ~ NH 之间 | 近低支撑买进，近高阻力卖出；或反向 T+0 |
| 跳空强势 | 价格 ≥ AH | 配合大盘/利好，于 AH 附近追买 |
| 跳空弱势 | 价格 ≤ AL | 配合大盘/利空，于 AL 附近追卖（做空）|

## 🚀 使用方法

1. **手动输入**：直接修改最高价、最低价、收盘价，CDP 点位自动更新。
2. **通过股票代码获取**：
   - 在上方“股票代码”输入框中输入 A 股代码（如 `600519`）。
   - 点击 **“获取最新数据”**，系统将自动填充最新交易日的高、低、收数据并计算点位。
3. **快捷选股（推荐）**：
   - 在“快捷选股区”输入股票代码，点击 **“添加快捷选股”**，系统自动获取股票名称并保存。
   - 点击已保存的股票标签，即可一键加载该股票的最新数据。
   - 每个标签右侧有 **✕** 按钮，可删除不需要的股票。
4. **清空代码**：点击“清空代码”按钮可清空当前输入的股票代码及状态提示。

## 📁 本地存储说明

- 快捷股票列表会保存在浏览器的 `localStorage` 中，键名为 `cdp_quick_stocks`。
- 清除浏览器缓存或手动删除数据会导致快捷列表丢失。

## 🔌 数据接口

- **行情数据**：腾讯财经 API  
  `https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param={code},day,,,10,qfq`
- **股票名称**：腾讯财经基础接口  
  `https://qt.gtimg.cn/q={code}`（返回 GBK 编码）

> 仅支持 A 股主板、创业板、科创板（6 开头沪市，0/3 开头深市）。

## 🛠️ 技术栈

- 原生 HTML5 + CSS3 + JavaScript（ES6+）
- 无任何外部依赖，单文件即开即用
- 响应式布局，支持手机/PC

## 📝 注意事项

- 获取的数据为**最新完整交易日**的日 K 线数据（不复权或前复权）。
- 当日盘中数据无法获取，请使用收盘后统一计算。
- 若某股票当日停牌，则获取的是最近一个有交易的交易日数据。
- CDP 指标属于逆市短线工具，请结合其他技术分析使用，控制风险。
