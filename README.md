# 📊 CDP 逆市操作指标

一个专业的 A 股日内交易辅助工具，基于前一完整交易日的最高价、最低价、收盘价计算经典 CDP 支撑/阻力位，并提供快捷选股、历史记录回溯、深色金融看板风格。适用于短线逆市交易策略参考。

---

## ✨ 功能特性

- **智能数据获取**：输入 A 股代码，自动获取**上一完整交易日**（昨日）的高、低、收数据，杜绝盘中不完整数据干扰。每次刷新页面或跨日打开，自动更新至最新交易日数据。
- **CDP 关键位计算**：实时计算 AH（最高阻力）、NH（近高阻力）、CDP（中枢值）、NL（近低支撑）、AL（最低支撑）。
- **深色金融看板主题**：专业级暗色界面，橙金色与青蓝色搭配，专注、护眼。
- **快捷选股区**：最多保存 10 只常用股票，自动获取股票名称，点击即可加载数据。
- **历史查询记录**：每次成功获取数据自动保存（股票代码、名称、日期、价格、CDP 关键位）。**自动去重**：同一股票同一交易日只保留最新一条。最多保留 15 条，支持清空。
- **自动实时计算**：手动修改任意价格字段，CDP 点位立即更新。
- **本地存储**：快捷股票列表和历史记录保存在浏览器 localStorage 中，数据不丢失。

## 🧮 CDP 计算公式

| 指标 | 公式 | 说明 |
|------|------|------|
| CDP | (H + L + 2×C) / 4 | 中枢值，多空分界线 |
| AH | CDP + (H - L) | 最高阻力（突破后强势） |
| NH | 2×CDP - L | 近高阻力（箱体上沿） |
| NL | 2×CDP - H | 近低支撑（箱体下沿） |
| AL | CDP - (H - L) | 最低支撑（跌破后弱势） |

> H / L / C 分别为**上一完整交易日**的最高价、最低价、收盘价。

## 📈 逆市操作参考

| 市场状态 | 价格位置 | 操作建议 |
|---------|----------|----------|
| 箱体震荡 | NL ~ NH 之间 | 在 NL 附近买进，NH 附近卖出（或反向 T+0） |
| 向上跳空 | 价格 ≥ AH | 配合大盘/利好，在 AH 附近追买 |
| 向下跳空 | 价格 ≤ AL | 配合大盘/利空，在 AL 附近追卖（做空） |

## 🚀 使用指南

### 1. 手动输入数据
直接修改「昨日最高价(H)」、「昨日最低价(L)」、「昨日收盘价(C)」输入框，CDP 点位自动重算。

### 2. 通过股票代码自动获取
- 在「股票代码」输入框输入 A 股代码（如 `600519`、`000858`）。
- 点击 **「获取昨日数据」**，系统将自动获取上一完整交易日的高、低、收数据并计算 CDP，同时保存到历史记录。

### 3. 快捷选股（推荐）
- 在「快捷选股区」输入股票代码，点击 **「➕ 添加快捷选股」**，自动获取股票名称并保存。
- 点击已保存的股票标签，一键加载该股票的最新昨日数据。
- 每个标签右侧有 **✕** 按钮，可删除。

### 4. 历史记录回溯
- 每次成功获取数据后，会自动保存记录（股票名称、日期、价格、CDP 关键位）。
- **同股票同日期只会保留最新一条查询记录**，避免重复。
- 点击任意历史记录，自动填充数据并重新计算 CDP。
- 可点击 **「清空全部」** 清除所有历史。

### 5. 清空输入
点击 **「清空代码」** 可清空当前股票代码及状态提示。

## 🔌 数据接口

- **行情数据**：腾讯财经 API  
  `https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param={code},day,,,10,qfq`
- **股票名称**：腾讯财经基础接口（GBK 编码）  
  `https://qt.gtimg.cn/q={code}`

> 仅支持 A 股主板、创业板、科创板（6 开头沪市，0/3 开头深市）。

## 📁 本地存储说明

| 存储键名 | 内容 | 容量 |
|----------|------|------|
| `cdp_quick_stocks` | 快捷股票列表（代码+名称） | 最多 10 条 |
| `cdp_history_records` | 历史查询记录（含完整价格及 CDP 位） | 最多 15 条，自动去重 |

## 📝 注意事项

1. **数据始终为上一完整交易日**（昨日），不受盘中实时价格影响。适合在每个交易日开盘前或收盘后使用。
2. **历史记录自动去重**：同一股票同一交易日只会保存最新一次查询结果。
3. **跨日自动更新**：页面每次加载时都会请求最新接口，获取最新的昨日数据（过零点后刷新页面即可）。
4. 若某股票停牌，则获取的是最近一个有交易的交易日数据。
5. CDP 指标属于逆市短线工具，请结合其他技术分析及风险控制使用。
